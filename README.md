# Michalis Portfolio

A professional portfolio website showcasing skills, experience, and projects in web development, cybersecurity, software engineering, and immersive technology.

## Features

- **Responsive Design** - Mobile-first approach with clean, professional styling
- **Multi-page Layout** - Home, About Me, and Contact sections
- **Skills Section** - 2x2 grid showcasing expertise in 4 key areas:
  - Web Development
  - Cybersecurity
  - Immersive Tech & VR
  - Software Engineering
- **Modern Header** - Centered navigation with branded logo
- **Compact Footer** - Professional footer with contact info, social links, and skills badges

## Pages

- **index.html** - Home page with introduction and highlights
- **bio.html** - About Me page with detailed skills and goals
- **contact.html** - Contact information page

## Technologies Used

- HTML5
- CSS3
- Responsive Web Design

- Primary Gold: #D4AF37

## Usage

Open `index.html` in a web browser to view the portfolio.
